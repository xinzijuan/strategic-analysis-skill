# Strategic Analysis Skill

围绕特定战略问题进行需求理解、逻辑拆解storyline，通过案头研究等形式给出独特战略判断，形成科技公司偏好的冷静深度Word文档。

## 功能简介

基于麦肯锡方法论，提供专业级战略分析服务。通过金字塔逻辑拆解、MECE原则论证、数据驱动决策，输出结构清晰的Word文档，适用于S1高管汇报、业务决策、竞品分析等场景。

## 适用场景

- **行业研究**：市场规模、增长趋势、竞争格局分析
- **市场调研**：用户画像、需求分析、机会评估
- **竞品分析**：竞对能力、定位差异、市场份额对比
- **商业决策**：战略选择、投资决策、合作/并购评估
- **产品/技术路线**：产品规划、技术栈选择、发展路径
- **企业战略**：增长策略、转型策略、竞争应对

## 不适用场景

- 需要实时交易数据的分析
- 需要访问私有数据（除非用户提供）
- 需要预测未来的具体数值（如股价、销量）
- 需要法律、财务等高度专业领域的专业判断

## 快速开始

1. **触发技能**：对话中输入"战略分析"或相关关键词
2. **提供战略问题**：描述你想解决的核心问题
3. **确认议题树**：确认需要分析的子问题
4. **等待研究**：AI进行案头研究和深度分析
5. **获取输出**：获得Storyline和对应的Word文档

## 工作原理

### 分析方法论

本技能采用麦肯锡经典方法论：

1. **金字塔原理**：自上而下结构化思维，结论在顶端，论据在下方
2. **MECE原则**：确保论证相互独立、完全穷尽
3. **假设驱动**：先提出假设，再通过数据验证
4. **Case Study**：对关键判断进行案例式验证

### 执行流程

```
前置确认 → 议题拆解 → 案头研究 → 假设验证 → 输出Storyline → 生成Word文档
```

## 输出规范

### Word文档结构

| 章节 | 内容 |
|-----|------|
| 封面 | 标题、副标题、日期、汇报人 |
| 核心判断 | 1-2个核心观点 + 数据概览 |
| 市场分析 | TAM/SAM/SOM + 竞争格局 |
| 核心论证 | 完整论证逻辑链 + 论据 |
| 风险分析 | 风险矩阵 + 应对建议 |
| 战略建议 | 优先级排序 + 详细理由 |
| 行动路线图 | 阶段规划 + 预期成果 |
| 附录 | 数据来源汇总 |

### 质量标准

- **内容全面性**：覆盖所有关键维度，不遗漏重要论据
- **逻辑合理性**：论证逻辑链清晰，因果关系明确，MECE原则
- **观点独到性**：提供独特洞察，避免泛泛而谈，给出明确判断
- **数据支撑**：每个关键判断有具体数据支撑，标注来源

## 依赖要求

- **数据搜索能力**：需要联网搜索功能
- **文件读取能力**：需要读取参考文档和模板
- **分析时长**：复杂战略分析可能需要多次交互

## 故障排除

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| 技能未触发 | 关键词不匹配 | 尝试其他触发词：市场调研、竞品分析、商业决策 |
| 分析不够深入 | 输入问题过于模糊 | 提供更具体的分析范围和维度 |
| 输出不符合预期 | 风格偏好未匹配 | 明确说明需要的风格（如更简洁/更详细/更数据驱动） |
| 数据无法获取 | 关键数据缺失 | 提供部分已知数据，或明确使用假设 |
